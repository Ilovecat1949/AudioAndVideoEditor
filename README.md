# 音视频编辑器
## 简介
这是一款基于FFmpeg的Android音视频编辑器，目标是在Android上提供大多数FFmpeg的功能。目前支持视频重编码和获取音视频信息功能。
## 基本功能
### 获取音视频信息
选择音视频文件，可以获取音视频的分辨率、帧率、码率、时长、声道数等信息
### 音视频重编码
选择音视频文件，可以设置编码的帧率、码率等信息，重新编码音视频文件。
